位掩码技巧

原理讲解

用于快速判断是否有重复出现的字符。限制条件为必须均为小写字母构成(不是必须)，则可以使用位掩码技巧：用一个(长度为26比特位，1表示含有该字母，0表示不含有)数存储字符串包含字母情况，则判断两个字符串可以通过两个位掩码数与计算是否为0(num1 & num2 == 0 ?)。

如何设置掩码的第 n 位？使用标准的位操作：n_th_bit = 1 << n。

如何计算一个单词的位掩码？遍历单词的每个字母，计算该字母在掩码中的位置 n = (int)ch - (int)'a'，然后创建一个第 n 位为 1 的掩码 n_th_bit = 1 << n，通过或操作将该码合并到位掩码中 bitmask |= n_th_bit。

LeetCode例题

代码如下：

func maxProduct(words []string) int {
    // 双循环
    // 判断两个单词是否有相同字母，用位掩码
    result, n := 0, len(words)
    bitsWords := make([]int, n)
    for i := 0; i < n; i++ {
        length := len(words[i])
        temp := 0
        for j := 0; j < length; j++ {
            t := int(words[i][j] - 'a')
            temp |= 1 << t
        }
        bitsWords[i] = temp
    }
    for i := 0; i < n - 1; i++ {
        for j := i + 1; j < n; j++ {
            if bitsWords[i] & bitsWords[j] == 0 {
                result = max(result, len(words[i]) * len(words[j]))
            }
        }
    }
    return result
}

func max(x, y int) int {
    if x > y {
        return x
    }
    return y
}
